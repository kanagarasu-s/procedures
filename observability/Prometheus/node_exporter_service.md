### 4Ô∏è‚É£ Create a Systemd Service File
#### Create the service unit:


#### Paste the following:

```sh

After=network.target

Group=node_exporter
Type=simple
ExecStart=/usr/local/bin/node_exporter

[Install]
WantedBy=default.target
```
Save and exit (:wq in Vim).
